Chapter 00: Project Kickstart & Vision

0.3: Project Creation & Folder Setup

1. Run this command in your terminal:

flutter create habit_tracker

2. Open the `habit_tracker` folder. Inside lib, create a `controllers` folder. Our structure is now:

lib/
├── controllers/
├── models/
├── screens/
├── widgets/
└── main.dart

Why do we create this folder structure?

Organizing code makes it easier to find things and work efficiently. This structure separates our code based on its purpose, which is a core principle of good software design.
