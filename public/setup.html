Windows Setup Guide

A comprehensive guide to setting up your Windows machine for Flutter development.

01.

Install Prerequisites

A

Install Git for Windows

Git is a version control system required by Flutter. Download it from the official site.

Download Git
B

Install Android Studio

Android Studio provides the Android SDK, emulator, and other essential tools.

Download Android Studio
02.

Install Flutter SDK

①

Download the SDK

Get the latest Flutter SDK stable release from the official documentation website.

Flutter Install Docs
②

Extract to a Permanent Location

Extract the ZIP archive to a location like C:\flutter.

Avoid directories like C:\Program Files that require administrator privileges.
03.

Configure Environment Path

①

Open Environment Variables

In the Start Menu, search for "env" and select "Edit the system environment variables".

②

Update the Path Variable

Select "Path" under User or System variables, click "Edit...", then "New", and add the full path: C:\flutter\bin

04.

Verify Setup

①

Run Flutter Doctor

Open a **new** terminal window and run flutter doctor. This command checks your setup and reports its status.

②

Accept Android Licenses

As prompted, run flutter doctor --android-licenses and accept all licenses by pressing 'y'.

Setup Complete!

Once `flutter doctor` reports no issues, you are ready to start building Flutter apps.
