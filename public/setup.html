Setup & Foundations

Welcome to the first step in your Flutter journey! This guide will walk you through installing the Flutter SDK, setting up your editor, and running your first app. Follow these steps carefully to ensure a smooth development experience.

Installing the Flutter SDK

The first step is to get the Flutter Software Development Kit (SDK) onto your machine. The official Flutter documentation is the best source for this, but here's a summary.

For Windows:

  1. Download the latest stable release of the Flutter SDK from the official Flutter website.
  2. Extract the zip file and place the contained flutter folder in a permanent location (e.g., C:\flutter). Do not install it in a directory that requires elevated privileges like C:\Program Files\.
  3. Update your Path environment variable to include the full path to C:\flutter\bin.
  4. Open a new command prompt and run flutter doctor. This command checks your environment and displays a report of the status of your Flutter installation.

For macOS:

  1. Download the latest stable release for macOS (select the appropriate Apple Silicon or Intel chip version) from the official Flutter website.
  2. Extract the file in a desired location, e.g., ~/development/flutter.
  3. Add the flutter tool to your path: export PATH="$PATH:`pwd`/flutter/bin".
  4. Run flutter doctor to check for any dependencies you need to install.

Creating Your First App

Once flutter doctor reports no major issues, you're ready to create a new project.

  1. Navigate to the directory where you want to create your project.
  2. Run the following command in your terminal:
flutter create my_first_app

This will create a new directory called my_first_app with a simple demo application.

Running the App

To run the app, you need an emulator, simulator, or a physical device connected.

  1. Change into the new project directory: cd my_first_app
  2. Run the app with the following command:
flutter run

After a few moments, you should see the Flutter demo app launch on your selected device. Congratulations, you're now a Flutter developer!
