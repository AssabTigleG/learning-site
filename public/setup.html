Windows Setup Guide

This comprehensive guide will walk you through every step of setting up your Windows machine for Flutter development.

Step 1: Install Prerequisites

A. Install Git for Windows

Git is required by Flutter to manage its dependencies. Download it from the official site.

B. Install Android Studio

Android Studio provides the Android SDK, emulator, and other essential command-line tools.

1. Download from the official Android developer website.

2. During installation, choose the Standard install type.

3. Once installed, open Android Studio and go to the SDK Manager.

4. Under the SDK Tools Tab, select the required options, and press Apply.

Step 2: Install the Flutter SDK

1. Download the Flutter SDK from the official website. The VS Code extension page provides a convenient link.

2. Once downloaded, extract the ZIP archive to a permanent location, for example: C:\flutter. Do not use a directory like C:\Program Files that requires administrator privileges.

Step 3: Configure Environment Path

You must add the Flutter tool to your system's path to run Flutter commands from any terminal.

1. In the Windows Start Menu, search for "env" and select "Edit the system environment variables".

2. In the System Properties window, click the "Environment Variables..." button.

3. Under "User variables" (or "System variables"), find "Path", select it, and click "Edit...".

4. In the "Edit" window, click "New", add C:\flutter\bin, and click OK.

Step 4: Final Setup & Verification

1. Open a **new** Command Prompt window.

2. Run flutter doctor. This command checks your setup and reports its status.

3. As prompted, run flutter doctor --android-licenses and accept all licenses.

4. Install the Flutter extension for Visual Studio Code.

Once `flutter doctor` reports no issues, you are ready to start building Flutter apps! Congratulations!
