Project: Simple Habit Tracker

Module 1: Building the Main Screen

Step 1.1: Create the Habit Model

Define the data structure for a single habit.

First, we need to create a simple Dart class in `lib/models/habit.dart` that will hold the information for each habit, such as its name and whether it's completed.

class Habit {
  final String name;
  bool isCompleted;

  Habit({required this.name, this.isCompleted = false});
}

Step 1.2: Build the Static List View

Layout the main screen with placeholder data.

Now, let's modify `lib/screens/main_screen.dart`. We will use a `ListView.builder` to display a static list of habits using our new `Habit` model.

// ... inside your StatefulWidget's build method
ListView.builder(
  itemCount: 3, // Placeholder count
  itemBuilder: (context, index) {
    return ListTile(
      title: Text('Habit placeholder $index'),
      trailing: Checkbox(value: false, onChanged: (val) {}),
    );
  },
);

Why use `ListView.builder`?

The `.builder` constructor is highly performant because it only builds the list items that are currently visible on screen, recycling them as the user scrolls. This is essential for long lists.
